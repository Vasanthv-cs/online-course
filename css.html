CSS Course

CSS (Cascading Style Sheets) is a language used to describe the presentation of a document written in HTML or XML.

CSS Examples

CSS Example 1:
/* CSS Styling */ body { background-color: #f0f8ff; color: #333; font-family: Arial, sans-serif; } h1 { color: #32CD32; text-align: center; } p { font-size: 1rem; }

This is Example 1 Output

This paragraph is styled using CSS with a light blue background and centered green heading.

CSS Example 2:
/* Another CSS Example */ div.container { width: 80%; margin: 0 auto; padding: 20px; border: 3px solid #1E90FF; background-color: #fff; }

This is a div with 80% width, centered with blue borders and padding.

CSS Example 3:
/* CSS Box Model Example */ div.box { width: 300px; padding: 20px; border: 5px solid #FFD700; margin: 10px auto; background-color: #f0f8ff; }

This is a box with padding and margin, styled using CSS box model.

CSS Example 4:
/* CSS Flexbox Example */ div.flex-container { display: flex; justify-content: space-around; background-color: #f0f8ff; } div.flex-container div { background-color: #32CD32; color: white; padding: 15px; margin: 10px; border-radius: 10px; }
